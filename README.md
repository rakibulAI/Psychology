# 🧠 মনোবিজ্ঞান মাস্টারক্লাস

> মানব মনের গভীরে প্রবেশ করো — সম্পূর্ণ বাংলায়, বিনামূল্যে।

---

## 📖 প্রকল্পটি কী?

**মনোবিজ্ঞান মাস্টারক্লাস** হলো একটি সম্পূর্ণ ইন্টারেক্টিভ HTML বই, যেখানে মানব মনোবিজ্ঞানের ৬০টিরও বেশি গুরুত্বপূর্ণ টপিক বাংলায় বিস্তারিতভাবে আলোচনা করা হয়েছে। প্রতিটি বিষয় সহজ ভাষায়, বাস্তব উদাহরণ সহ উপস্থাপন করা হয়েছে — যাতে একজন সাধারণ পাঠকও নিজেকে একজন মনোবিজ্ঞানীর মতো করে চিন্তা করতে পারেন।

এটি কোনো ভারী একাডেমিক পাঠ্যবই নয়। এটি একটি **প্র্যাকটিক্যাল গাইড** — জীবনে, সম্পর্কে, কাজে, এবং নিজেকে বোঝার জন্য।

---

## ✨ বৈশিষ্ট্যসমূহ

- 🌐 **দ্বিভাষিক সমর্থন** — বাংলা ও ইংরেজি উভয় ভাষায় পড়া যায়, এক ক্লিকেই ভাষা পরিবর্তন করা যায়
- 🔍 **লাইভ সার্চ** — যেকোনো টপিক মুহূর্তের মধ্যে খুঁজে বের করো
- 📊 **রিডিং প্রগ্রেস ট্র্যাকার** — কতগুলো টপিক পড়েছো তা দেখা যায়
- 🎨 **ডার্ক থিম ডিজাইন** — চোখের আরামদায়ক, আধুনিক UI
- 📱 **মোবাইল ফ্রেন্ডলি** — যেকোনো ডিভাইসে সুন্দরভাবে দেখা যায়
- ⚡ **সম্পূর্ণ অফলাইন** — একটিমাত্র HTML ফাইল, ইন্টারনেট ছাড়াও চলে
- 🔝 **স্ক্রল টু টপ বাটন** — দ্রুত নেভিগেশন
- 🃏 **অ্যাকর্ডিয়ন কার্ড** — প্রতিটি টপিক expand/collapse করা যায়

## 📚 বিষয়বস্তু — ৮টি প্রধান বিভাগ

### ০১ · প্রভাব ও ম্যানিপুলেশন
মানুষকে প্রভাবিত করার মূল কৌশলগুলো — ভালো ও খারাপ উভয় দিক সহ।
- Reciprocity (পারস্পরিক বাধ্যবাধকতা)
- Social Proof (সামাজিক প্রমাণ)
- Scarcity Principle (দুষ্প্রাপ্যতার নীতি)
- Authority Bias (কর্তৃত্বের প্রভাব)
- Foot-in-the-Door Technique
- Intermittent Reinforcement
- Liking Principle
- Commitment & Consistency

### ০২ · Cognitive Biases — চিন্তার ফাঁদ
আমাদের মস্তিষ্ক যে ভুলগুলো বারবার করে।
- Confirmation Bias
- Anchoring Effect
- Dunning-Kruger Effect
- Sunk Cost Fallacy
এবং আরও অনেক...

### ০৩ · আবেগের মনোবিজ্ঞান
কীভাবে আবেগ আমাদের সিদ্ধান্ত ও আচরণ নিয়ন্ত্রণ করে।
- Emotional Contagion (আবেগের ছোঁয়াচে প্রভাব)
- Fear Appeal (ভয়ের আবেদন)
এবং আরও...

### ০৪ · সামাজিক মনোবিজ্ঞান
মানুষ দলে কীভাবে আচরণ করে এবং কেন।
- Bystander Effect
- Groupthink
- Fundamental Attribution Error
এবং আরও...

### ০৫ · অনুপ্রেরণার মনোবিজ্ঞান
মানুষ কেন কাজ করে, কীভাবে নিজেকে ও অন্যকে motivated রাখা যায়।
- Maslow's Hierarchy of Needs
- Intrinsic vs Extrinsic Motivation
এবং আরও...

### ০৬ · ডার্ক সাইকোলজি
মনোবিজ্ঞানের অন্ধকার দিক — শেখো যাতে নিজেকে রক্ষা করতে পারো।
- Gaslighting
- Love Bombing
- Narcissism
এবং আরও...

### ০৭ · থেরাপি ও নিরাময়
মানসিক স্বাস্থ্য রক্ষার বৈজ্ঞানিক পদ্ধতিগুলো।
- CBT (Cognitive Behavioral Therapy)
- Mindfulness
- Stoicism
এবং আরও...

### ০৮ · নিউরোসাইকোলজি
মস্তিষ্ক কীভাবে কাজ করে — বিজ্ঞানের আলোয়।
- Dopamine & Reward System
- Neuroplasticity
- Fight-or-Flight Response
এবং আরও...

## 🚀 কীভাবে ব্যবহার করবো?

এটি একটি সিঙ্গেল-ফাইল HTML প্রজেক্ট। ব্যবহার করতে মাত্র তিনটি ধাপ:

```bash
# ১. রিপোজিটরি clone করো
git clone https://github.com/rakibulAI/psychology-masterclass.git

# ২. ফোল্ডারে যাও
cd psychology-masterclass

# ৩. ফাইলটি ব্রাউজারে খোলো
# Windows: psychology-master.html ফাইলে ডাবল ক্লিক করো
# Mac/Linux: open psychology-master.html
```

অথবা সরাসরি ফাইলটি ডাউনলোড করে যেকোনো ব্রাউজারে (Chrome, Firefox, Edge) খোলো। **কোনো server বা installation লাগবে না।**
